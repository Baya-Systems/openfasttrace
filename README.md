# Installation

## Dependencies

OpenFastTrack needs a Java JDK 8 (or later) implementation and Maven as build tool.

## On Linux

### Ubuntu or Debian

    apt-get install openjdk-8-jre

## Development

Run `mvn test` to run unit tests.
